# Manuscript source

`manuscript.tex` is the latest IEEE conference-format proposal supplied by the
project team on 2026-07-10. It is kept beside the implementation so that model,
decision, telemetry, and analysis contracts can be reviewed against the study
methodology.

The source references `proposal_2_mockup.png`, which was not included with the
submitted LaTeX. Place that image in this directory before compiling. A TeX
distribution with `IEEEtran`, TikZ, and the packages listed in the preamble is
also required.

Repository code and generated experimental results remain the source of truth
for implemented behavior. Preliminary agronomic values and bibliography items
marked `REVIEW` in the manuscript still require field or source validation
before the final defense.
